🎚️ Frequency Map

Where each band lives and what to listen for.

Sub-bass

20-60 Hz
  • 808 fundamental
  • HP below 30Hz if muddy

Bass/Body

60-200 Hz
  • 808 punch, kick weight
  • Watch masking

Low-Mid (mud)

200-500 Hz
  • Vocal warmth @ 250Hz
  • Cut if boxy

Mids

500-2k Hz
  • Vocal body, snare
  • Honk ~800Hz

Upper Mids

2-6 kHz
  • Presence, de-ess zone

Highs

6-10 kHz
  • Air, cymbal shimmer

Extreme

10-20 kHz
  • Maag magic band
